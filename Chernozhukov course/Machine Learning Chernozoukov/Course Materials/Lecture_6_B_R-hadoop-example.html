Preliminaries

  • We assume that computer has an installation of hadoop as a single cluster (this means we shall use temporal distribution of computing power of a single machine).

  • This is useful even if we work with spacial clusters, since we typically need to write a code and test it on a single computer, before employing a cluster of computers.

  • The details on installing hadoop are given here: http://amodernstory.com/2014/09/23/installing-hadoop-on-mac-osx-yosemite/

It is a very painful process, so don’t do it yet, unless you really have to.

Launching Hadoop

  • If you installed hadoop using the steps above, you should be able to type in the terminal window “hstart”, which will launch the hadoop. The link http://amodernstory.com/2014/09/23/installing-hadoop-on-mac-osx-yosemite/ provides you with some troubleshooting steps, if you get stuck. In particular, you might have to run hadoop namenode -format, if you get “connections refused messages”.

  • Launch R-studio, open the .R script file “RhadopExample.R”

  • We then run chunks of code and display them in the slides.

With MapReduce

step 1: write data into dfs (distribute file system that hadoop is using)

XX<-cbind(y,X)
in.ols<-to.dfs(cbind(y,X))

Step 2: write the mapper

ols.map<- function(.,YXi) { 
  Yi = as.matrix(YXi[,1])
  Xi=YXi[,-1]
  c.keyval(keyval('XtX',list(t(Xi)%*%Xi)), 
           keyval('XtY',list(t(Xi)%*%Yi)))
}
#The mapper takes i-th chunk of (Y,X) 
# and produces X'X and X'Y for that chunk. 
#The formal output will be two key-value pairs:
# key_1 = 'XtX',  val_1 = X'X
# key_2 = 'XtY', val_2 X'Y
# The word "t" stands for the transpose.

Step 3: write the reducer

# Reducer operates on the KeyValueStore:
# ('XtX',[val_11,val_12,..val_1M])
# ('XtY',[va1_21,val_22,...,val_2M])
# We need to produce:
# t(X)%*%X = t(X1)%*%X1+t(X2)%*%X2+...+t(XM)%*%XM = val_11+..+val_1M
# t(X)%*%Y = t(X1)%*%y1+....=val_21+val_22+...+val_2M
# Thus reducer just does "+"

ols.reduce<-function(keys,vals) { keyval(keys,list(Reduce('+',vals)))}

# type help(Reduce); this function is a standard function in r

Step 4: call the map reduce

# step 4 call mapreduce
ols.out <- mapreduce(input = in.ols, map = ols.map, 
    reduce = ols.reduce, combine = TRUE, verbose = FALSE)
XtXXtY <- from.dfs(ols.out)
# XtXXtY has two key-value pairs: (1,XtX) and
# (2,XtY)
print(XtXXtY$key)  #show the two keys
## [1] "XtX" "XtY"
XtX = XtXXtY$val[[1]]  #extract values
XtY = XtXXtY$val[[2]]  #extract values
b_est.mr <- solve(XtX) %*% XtY

  hsg+cg+ad+mw+so+we+exp1+exp2+exp3+exp4

with hadoop

X <- model.matrix(fmla)
y <- lnw
YX <- cbind(y, X)
in.ols <- to.dfs(YX)
ols.out <- mapreduce(input = in.ols, map = ols.map, 
    reduce = ols.reduce, combine = TRUE, verbose = FALSE)
# XtXtY has two key-value pairs: (1,XtX) and
# (2,XtY)
XtXXtY <- from.dfs(ols.out)
print(XtXXtY$key)  #show the two keys
## [1] "XtX" "XtY"
XtX = XtXXtY$val[[1]]  #extract values
XtY = XtXXtY$val[[2]]  #extract values
b.est.mr <- as.vector(solve(XtX) %*% XtY)
names(b.est.mr) <- colnames(X)
